# Live Global weather and AQI Dashboard
### Final Project for Data Visualization and pre-processing for FY BTech E-group at COEP Technological university, Pune

## Abstract Summary
This was a group project to create a dashboard in Power BI with the help of publicly available datasets. We chose weather and aqi as thr topic for the dashboard to be cognizant of local and global weather, and provide a tangible way to view data with ease.

## Data Analytics using Power BI
* **Data Sources:** Kaggle, Visual Crossing API
* **Key Metrics:** Live city weather, temperature and aqi across years, humidity per city, AI-based Q&A

## How to Run It
1. Clone this repo.
2. Open `/reports/dashboard.pbip` in Power BI Desktop.
